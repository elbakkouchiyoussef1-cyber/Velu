Velu Beauty Studio Website

Overview

This project is a modern, responsive website created for a beauty studio.
The website presents the studio’s services, contact information, and provides access to an external booking platform.

The site is built as a static website using HTML, CSS, and JavaScript to ensure fast loading times and easy deployment on any hosting provider.

Features
	•	Responsive design for desktop, tablet, and mobile
	•	Modern beauty-focused layout
	•	Smooth scrolling animations
	•	Scroll-based reveal effects
	•	Service sections with detailed descriptions
	•	Overlay panels for additional service information
	•	Direct links to Instagram, email, and phone contact
	•	External booking platform integration

Technologies Used
	•	HTML5
	•	CSS3
	•	JavaScript
	•	GSAP animation library

Deployment

Testing with GitHub Pages

The project can be tested online using GitHub Pages.

Steps:
	1.	Upload the project to GitHub
	2.	Go to Repository Settings
	3.	Select Pages
	4.	Choose Deploy from branch
	5.	Select main / root

The site will then be available at:
https://elbakkouchiyoussef1-cyber.github.io/repository-name/
Production Hosting

The final version of the website is intended to be hosted on the client’s hosting provider.

Recommended setup:
	•	Static website hosting
	•	HTTPS enabled
	•	Domain connected through the hosting provider

External Services

The booking and payment functionality are handled by an external platform chosen by the client.
This website only links to that platform and does not process payments or bookings directly.
